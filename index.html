WP Store

WP Store is an independent plugin manager that lets you install and update Ilterra's proprietary WordPress plugins without relying on the official repository.

Loading…

Why WP Store?

  • Install and update Ilterra plugins directly.
  • Streamlined releases without external review queues.
  • Secure distribution for closed-source plugins.

You can find ZIP archives (for direct installation) with plugins in the releases.
